 # CILabProject
## CI and Jenkins Assignment

This project demonstrates the implementation of Continuous Integration (CI) concepts using Jenkins.

## Assignment Overview
The project covers theoretical understanding and practical implementation of CI using Jenkins, including installation, job configuration, pipelines, and plugin management.

## Features Implemented
- Jenkins installation and configuration on local system
- Freestyle Jenkins job with GitHub integration
- Multibranch Pipeline with branch-based build strategies
- Maven-based Java project with JUnit testing
- Jenkins architecture diagram and documentation

## Jenkins Jobs
- **Freestyle-CI-Lab**
- **Multibranch-CI-Lab** (main, feature, release branches)

## Repository Structure
- `src/` – Java source code and test cases
- `scripts/` – Build and deployment scripts
- `docker/` – Dockerfile for CI setup
- `docs/` – Screenshots, architecture diagram, and reports
- `Jenkinsfile` – Jenkins pipeline configuration
- `pom.xml` – Maven configuration file

## Tools & Technologies
- Jenkins (LTS)
- Git & GitHub
- Maven
- Java
- JUnit
